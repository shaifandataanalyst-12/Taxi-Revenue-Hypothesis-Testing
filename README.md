# Taxi Revenue Hypothesis Testing

## Project Overview
This project analyzes taxi trip data to investigate whether payment methods influence the total fare amount.

## Problem Statement
Understanding the relationship between payment method and fare amount can help identify potential revenue patterns and optimize driver payment strategies.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

## Dataset
https://drive.google.com/file/d/18XjmeyDf-xNK94c9th-EQ5KLRMt5_02S/view?usp=drive_link
Taxi trip dataset containing:
- Trip distance
- Payment type
- Fare amount
- Pickup location
- Drop-off location


## Analysis Performed
- Data cleaning and preprocessing using Pandas
- Exploratory data analysis
- Hypothesis testing (T-test)
- Data visualization

## Key Insights
- Credit card payments resulted in **higher average fares compared to cash payments**.
- Payment behavior had a measurable impact on total trip revenue.
- The analysis suggests potential incentives for encouraging digital payments.
